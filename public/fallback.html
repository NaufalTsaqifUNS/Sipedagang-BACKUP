Oops! Terjadi Masalah dengan Aplikasi

Aplikasi tidak dapat dimuat dengan benar. Berikut adalah beberapa kemungkinan penyebabnya:

Kemungkinan Masalah:

  • Koneksi ke server API tidak berhasil
  • Terjadi kesalahan saat memproses CSRF token
  • Browser Anda mungkin memblokir cookie yang diperlukan
  • Terjadi kesalahan JavaScript saat memuat aplikasi

Masalah Cookie

Pastikan browser Anda mengizinkan cookie dari domain ini. Periksa pengaturan privasi browser Anda.

Masalah Koneksi

Pastikan Anda memiliki koneksi internet yang stabil dan server API sedang berjalan.

Browser Tidak Kompatibel

Coba gunakan browser modern lain seperti Chrome, Firefox, atau Edge terbaru.
